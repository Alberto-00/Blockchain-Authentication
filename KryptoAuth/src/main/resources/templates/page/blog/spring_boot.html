Spring Boot

Sprint Boot è un framework basato su Java utilizzato per lo sviluppo rapido di applicazioni (per creare microservizi autonomi). Ha un supporto aggiuntivo per la configurazione automatica e il server delle applicazioni incorporato come Tomcat, Jetty, ecc...

I vantaggi di Spring Boot sono la facilità nel capire e sviluppare applicazioni spring. Spring Boot non è altro che un framework esistente con l'aggiunta di un server HTTP incorporato e di una configurazione di annotazione che semplifica la comprensione e velocizza il processo di sviluppo. Aumenta la produttività e riduce i tempi di sviluppo. Possiede una configurazione minima e non è necessario scrivere alcuna configurazione XML, sono necessarie solo alcune annotazioni per eseguire la configurazione.

Caratteristiche

Spring Boot consente agli sviluppatori di eseguire la stessa applicazione in ambienti diversi. Questo viene fatto con il supporto che fornisce per la configurazione esterna. Utilizza variabili di ambiente, file di proprietà, argomenti a riga di comando, file YAML e proprietà di sistema per menzionare le proprietà di configurazione richieste. Inoltre, l'annotazione @value viene utilizzata per accedere alle proprietà.

Quando un'applicazione Spring Boot viene eseguita come "Esegui come applicazione Java", Spring Boot avvia automaticamente il server Tomcat non appena rileva l'applicazione web.

La gestione delle dipendenze Spring Boot viene fondamentalmente utilizzata per gestire automaticamente le dipendenze e la configurazione senza che si specifichi la versione per nessuna di queste dipendenze.
